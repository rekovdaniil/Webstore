The JDBC webstore for demonstration purpose. Combination of JavaEE(Servlets and JSP) with Basic frontend interface. 

Main features:
-authorisation and autectification;
-new user registration;
-operations with DB instances (Users and Items).
-fully functioning chartlist and list of items avaliable for order. 
-usage of Blochain.info API in order to place an actual BTC exchange rate. 
- etc.
DB and Webapplication was hosted AWS server for demonstration purpose, but due to pretty high $ pricing
on AWS EC2 24/7 functioning - just local version with localDB avaliable for local server (Tomcat Server in my case).
